# Coffee Break Belém - Café Coworking ☕

Aplicativo de Cartão Fidelidade desenvolvido em **Python + Streamlit**.

## Recursos
- Cadastro de clientes
- Registro de compras com cálculo automático de carimbos
- Detecção quando o cliente atinge o limite de carimbos (balões + aviso)
- Link de WhatsApp para notificação rápida
- Resgate de prêmios
- Painel de administração (estatísticas e configuração de regras)

## Como executar
```bash
pip install -r requirements.txt
streamlit run app.py
```
